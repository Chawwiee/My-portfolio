Wellwave - School Counseling and Mental Health Support System


A school-based mental health support system designed to address the emotional, psychological, and social challenges faced by students within the EARIST community. As academic pressures and personal challenges increase, many students experience stress and anxiety that traditional educational systems are often unable to adequately support. Wellwave bridges this gap by integrating accessible and student-centered mental health services directly into the school environment.


The system provides a safe, confidential, and non-judgmental platform where students can openly discuss their concerns with trained school counselors. By emphasizing wellbeing-centered counseling, Wellwave helps reduce stress, promote emotional resilience, and encourage healthier coping mechanisms among students.


Designed as a holistic mental health intervention, Wellwave aims to improve awareness of mental health, enhance access to counseling services, and align emotional wellbeing with academic development. Although tailored specifically for EARIST and limited by resource availability and counselor capacity, the system highlights the importance of prioritizing mental health in education and contributes to the creation of a supportive school environment where emotional wellbeing is valued alongside academic success.

Interactive Information Zoo catalog using QR Code


The Interactive Information Zoo Catalog using QR Code is a system designed to enhance zoo visitors’ learning experience by providing instant access to accurate and organized animal information through mobile devices. QR codes are placed near animal enclosures, which visitors can scan using a custom-built mobile application. Once scanned, the system retrieves and displays detailed content such as animal descriptions, habitat, diet, conservation status, and images, eliminating the need for printed materials and manual guides.


From an Operating Systems perspective, the project demonstrates key OS concepts such as process management, file handling, memory usage, and device interaction. The mobile application handles user input and camera access, while the backend system manages data storage, retrieval, and concurrent access requests efficiently. The system emphasizes responsiveness, efficient resource utilization, and reliable data access, showcasing how operating system principles support real-time, interactive applications in a practical, real-world environment.

CCS Information and Room Tracking System


The CCS Information and Room Tracking System is a desktop-based application designed to modernize the administrative and academic operations of the College of Computing Studies at EARIST. Developed to replace inefficient manual logbooks and paper-based workflows, the system serves as a centralized platform that digitizes record-keeping, information dissemination, and resource monitoring.


The system functions through three core integrated modules to streamline college operations. An interactive kiosk interface allows students and faculty to search for real-time class schedules, faculty profiles, section details, and room assignments, effectively eliminating the need for physical bulletin boards. The platform replaces manual logbooks by digitally recording professor attendance in classrooms and tracking specific PC usage by students in computer laboratories. Furthermore, the system enhances security and management by integrating hardware sensors, such as RFID and laser beams, to validate professor presence, detect unauthorized room entry, and trigger alarms for security breaches. By automating data validation and centralizing records, the project aims to resolve approval bottlenecks, ensure accurate resource accountability, and create a seamless, paperless environment for the college

Student Information System


EduTrack is a comprehensive mobile Student Information System designed to streamline academic management through a modern, user-friendly interface. The platform features a dual-entry system for students and administrators, ensuring secure access to school data through a unified login and registration portal. Once authenticated, users interact with a central dashboard that provides a high-level summary of total enrollment, active classes, and upcoming tasks. This allows for immediate institutional oversight and quick navigation to specialized modules for managing the student body and curriculum.


The system excels in organization by maintaining detailed digital records for both students and academic schedules. The student directory offers reliable filtering capabilities, allowing users to sort records by specific courses, year levels, and sections to find information quickly. Similarly, the class management module handles the complexities of school scheduling by tracking course codes, semester details, and specific weekly time slots. This ensures that administrators can easily update the institutional timetable and maintain an accurate list of active subjects.


Beyond basic record-keeping, EduTrack facilitates academic productivity through a dedicated activities portal. This module acts as a digital ledger for assignments and quizzes, providing a space to input detailed descriptions, resource links, and strict deadlines. By centralizing academic requirements and schedules in one cohesive environment, the system bridges the communication gap between administration and students, ensuring that all essential institutional data is organized and easily accessible on the go.

Automated Document Classification and Management for School Accreditation Benchmarking


The Automated Document Classification and Management System is an advanced AI-driven platform specifically engineered to streamline the labor-intensive school accreditation benchmarking process. Traditionally, accreditation requires the manual sorting and classification of vast quantities of unstructured documents such as course syllabi, faculty credentials, and student work samples. This system modernizes that workflow by integrating a multi-stage digital pipeline that begins with an Optical Character Recognition engine to convert scanned PDFs or images into machine-readable text. To ensure high efficiency, the system focuses its analysis on the front page of documents where the most critical identifying information is typically located.


The core intelligence of the system relies on a fine-tuned BERT model for semantic classification alongside a Named Entity Recognition module. While the BERT classifier determines the document's category based on the overall context of the text, the NER module simultaneously extracts specific data points like names, dates, and academic terms. The system then employs a weighted scoring engine that combines results from the BERT classifier, NER validation, and a keyword analysis engine to achieve high accuracy. This allows the system to remain layout-independent, meaning it can correctly identify and categorize a document based on its actual content rather than relying on a rigid physical template.


Once a document is processed and validated, the system automatically generates a standardized filename and assigns the file to one of ten predefined accreditation folders. This automated organization facilitates seamless storage and rapid retrieval for evaluators. Built to be adaptable, the platform also includes an administrative interface that allows institutions to add new document types or adjust folder constraints as accreditation standards evolve. By replacing manual sorting with this integrated natural language processing architecture, the system significantly reduces administrative workloads, minimizes human error, and improves the overall reliability of institutional data management.
